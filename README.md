# AI Career Guidance System

## 📖 Project Description
The AI Career Guidance System is a Python-based application that helps students explore suitable career paths based on their interests. It provides career recommendations and relevant course suggestions in three domains:
- Artificial Intelligence
- Data Science
- Software Development

## ✨ Features
- User-friendly command-line interface
- Career recommendations based on user interest
- Course recommendations for each career domain
- Menu-based options:
  - Career Recommendations
  - Recommended Courses
  - Both Careers and Courses
- Modular Python program using separate files

## 🛠️ Technologies Used
- Python 3
- GitHub

1. Open the project in VS Code.

```bash
python main.py
```
